## Image Optimization Guidelines

### Technical Specifications
- **Format**: JPG for photos, PNG for graphics with transparency
- **Size**: 1200px width maximum for web use
- **Quality**: 80-85% compression for balance of quality/file size
- **Alt Text**: Always include descriptive alt text for accessibility

### Unsplash URL Parameters
For dynamic resizing and optimization, use Unsplash's URL parameters:
```
https://images.unsplash.com/photo-[ID]?ixlib=rb-4.0.3&auto=format&fit=crop&w=[WIDTH]&h=[HEIGHT]&q=80
```

**Example**:
```
https://images.unsplash.com/photo-1606992476431-4ad49c0a3ee3?ixlib=rb-4.0.3&auto=format&fit=crop&w=1200&h=400&q=80
```

## Implementation Best Practices

### README.md Guidelines
1. **Hero Images**: Use 1200x400 banner format for section headers
2. **Module Images**: 600x300 for individual module showcases
3. **Center Alignment**: Use `<div align="center">` for professional presentation
4. **Captions**: Include italicized descriptions below images

### Website Integration
1. **Lazy Loading**: Implement for performance
2. **Responsive Design**: Ensure images scale properly on mobile
3. **Consistent Style**: Match image dimensions and aspect ratios
4. **Brand Colors**: Consider overlays that match Trident Systems branding

## Legal Considerations

### Unsplash License
- Free for commercial use
- No attribution required (but appreciated)
- Cannot be sold as standalone product
- Can be modified and distributed

### Pexels License
- Free for commercial use
- No attribution required
- Cannot be sold without significant modification
- Cannot distribute on stock photo platforms

### Best Practices
1. **Always verify license** before using any image
2. **Keep records** of image sources and licenses
3. **Consider attribution** even when not required
4. **Avoid trademarked content** in backgrounds
5. **Respect model releases** for people in photos
